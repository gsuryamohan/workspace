# Sample Developer Guide

A GitHub Wiki–style developer guide with multiple pages.

## Structure

| File | Purpose |
|------|---------|
| `Home.md` | Index and entry point |
| `_Sidebar.md` | Sidebar navigation (GitHub Wiki convention) |
| `Chapter-1.md` | Chapter 1: Getting Started |
| `Chapter-2.md` | Chapter 2: Architecture |
| `Chapter-3.md` | Chapter 3: API Reference |
| `Chapter-4.md` | Chapter 4: Configuration |
| `Chapter-5.md` | Chapter 5: Troubleshooting |

## Usage

- **GitHub Wiki:** Copy these files into a repo's wiki (e.g. `your-repo.wiki`).
- **Local:** Open `Home.md` and use the links to navigate.
- **Docs site:** Use with MkDocs or similar; add these files to the docs folder.
